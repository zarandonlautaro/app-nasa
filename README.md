# App NASA

Is a React project for dealing images from API of [NASA](https://api.nasa.gov/) .

## Installation

Use the package manager like npm or yarn

```bash
npm i
```
```bash
yarn
```

## Usage

```bash
npm run start
```
```bash
yarn start
```
